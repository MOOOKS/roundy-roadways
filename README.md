# Roundy Roadways

## Project Overview
Roundy Roadways is an incremental car simulation game built in Unity for the 2025 GMTK Game Jam. The game focuses on building circular road lanes, upgrading speed, and buying cars. I implemented dynamic music and scalable game progression.

## Demo
Playable demo: https://moooks999.itch.io/roundy-roadways

## Tools Used
- Unity (C#)
- FL Studio (music production)
- Photoshop, Piskel (art)
- LeanTween, Easy Text Effects (Unity plugins)

## Notes
All core gameplay systems and code were written by me. Art and audio were mostly created by me, with some musical collaboration.
